This is portfolio theme for me but you're welcome to make suggestions. It is original portfolio desgin. When I create blog theme for blog platform, theme will be based on this desgin. I try to something new in this website to keep up to date on Front-End technology or methodology.

私のポートフォリオやブログ用のテーマにになりますがいつでもこのサイトを良くするための提案は歓迎です。オリジナルのデザインで、他のブログプラットフォームを使ってそこでのテーマを作るときのベースになります。また、このサイトではFront-Endに関する新しいことにどんどん挑戦していきます。

このテーマは https://github.com/sotayamashita/cookbook で管理されています。
